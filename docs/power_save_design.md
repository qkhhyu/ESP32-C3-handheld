# 屏幕节能模式 开发设计文档

## 1. 需求概述

- **夜间熄屏**：18:30 之后自动关闭屏幕（关背光 + 显示屏休眠）
- **触摸唤醒**：熄屏状态下触摸屏幕可点亮，持续 1 分钟后自动再次熄屏
- **早晨自动亮起**：8:50 自动亮屏，常亮至 18:30
- **总开关**：在设置 App 中提供开关，可随时启用/禁用此功能
- **状态持久化**：开关状态保存在 NVS 中，断电重启后保持

## 3. 新增文件

### 3.1 `main/power_mgmt.h` — 电源管理模块接口

| 函数 | 说明 |
|---|---|
| `power_mgmt_init(panel_handle)` | 初始化：绑定 LCD 句柄、读取 NVS 开关状态、创建管理任务 |
| `power_mgmt_set_enabled(bool)` | 设置节能模式开关，写入 NVS |
| `power_mgmt_is_enabled()` | 查询当前开关状态 |
| `power_mgmt_notify_touch()` | 触摸事件通知（用于熄屏唤醒） |
| `power_mgmt_set_brightness(duty)` | 同步当前亮度值（恢复亮屏时使用） |

### 3.2 `main/power_mgmt.c` — 电源管理模块实现

#### 状态机设计

```
                    ┌──────────────┐
          daytime   │              │   night time
      ┌─────────────┤  SCREEN_ON   ├──────────────┐
      │             │              │              │
      │             └──────────────┘              │
      │                                           ▼
      │             ┌──────────────┐     ┌────────────────┐
      │   enable    │              │     │                │
      └─────────────┤  SCREEN_OFF  │     │ SCREEN_TEMP_   │
                    │              │     │ WAKE           │
                    └──────┬───────┘     │ (触摸点亮1分钟) │
                           │             └───────┬────────┘
                           │ 触摸唤醒             │
                           └────────────────────►│
                                     60秒超时后返回 SCREEN_OFF
```

#### 时间窗口判断

```c
// 转换为当天的分钟数判断
int minutes = hour * 60 + minute;

// 夜间: 18:30(1110) ~ 次日 8:50(530)
bool is_night = (minutes >= 1110 || minutes < 530);

// 白天: 8:50(530) ~ 18:30(1110)
bool is_daytime = (minutes >= 530 && minutes < 1110);
```

#### NVS 存储

| 命名空间 | 键 | 类型 | 说明 |
|---|---|---|---|
| `power_mgmt` | `enabled` | `uint8` | 0=禁用, 1=启用 |

#### 任务逻辑（每秒执行）

```
1. 若功能禁用 → 确保屏幕亮起，跳过后续
2. 若时间未同步 → 确保屏幕亮起，跳过后续
3. 若当前为白天 → 确保屏幕亮起，跳过后续
4. 若当前为夜间：
   - SCREEN_ON → 关闭屏幕
   - SCREEN_TEMP_WAKE → 检查是否满60秒，满则关闭屏幕
   - SCREEN_OFF → 保持熄屏
```

## 5. 关键设计决策与边界情况

| 场景 | 处理方式 |
|---|---|
| **NTP 时间未同步** | `time(NULL) < 1700000000` 视为未同步，屏幕保持常亮 |
| **用户在夜间关闭节能开关** | 立即亮屏，进入常亮模式 |
| **用户在白天开启节能开关** | 屏幕保持亮起，等到 18:30 才首次熄屏 |
| **夜间触摸唤醒后再次触摸** | 刷新 60 秒倒计时（重新计时 1 分钟） |
| **断电重启** | 读取 NVS 中保存的开关状态 |
| **亮度恢复** | 亮屏时恢复到用户在设置中设定的亮度值 |

## 6. 功耗分析（估算）

| 状态 | 背光 | 显示控制器 | 说明 |
|---|---|---|---|
| 常亮模式 | PWM 10%~80% | Active | 当前默认行为 |
| 夜间熄屏 | OFF | Sleep | 背光关闭 + ST7789 休眠 |
| 触摸唤醒 | 用户设定值 | Active | 持续 60 秒 |
